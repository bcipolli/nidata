.. container:: index-paragraph

    Nidata is a Python module for **fast and easy access to public 
    NeuroImaging** data.

    In addition to documentation and easy downloads of the data,
    each dataset comes with at least one example of how to use it.

    All this, and the package has zero package-level dependencies.
    Individual datasets and examples have dependencies, but if you
    have pip installed, they will be installed as needed, automagically.
